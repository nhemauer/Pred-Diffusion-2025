The files in this directory can be used to reproduce the results of the analysis of the first year in which the policy was adopted. 

It is assumed that each R script is run with the working directory set to the top-level directory. 

There is one core analysis script. Before this is run, the main analyses for Boehmke, Boushey, and Karch need to be run.

1. minyr_effect_plot.R: Makes plots of the coefficients for minimum year.
